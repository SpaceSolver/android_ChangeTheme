# android_ChangeTheme

## Themeの継承について
AppCompatActivityを継承している場合は、ThemeにTheme.AppCompat  
またはTheme.AppCompatを含んだThemeを指定しないとErrorとなる。

## ツールバー（アクションバー）について  
　OSの種類で同じ個所に表示されるバーの種類が違う。  
　そのため、styleで色を変えたりしたのに反映されず、ハマってしまった。  
　今は、ツールバーのタグが反映されるようだ。
